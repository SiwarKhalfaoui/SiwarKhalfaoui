## 🙋‍♀️ About Me

I'm a 3rd-year Software Engineering student at **ESPRIT School of Engineering**
who genuinely enjoys the full stack — I'm as comfortable designing a REST API
as I am building the interface that consumes it.

I've worked across multiple stacks which taught me that good software isn't about the framework,
it's about clean architecture, readable code, and solving the right problem.

- 🎓 Software Engineering · **ESPRIT**, Tunisia
- 🔨 I build things end-to-end — backend, frontend, deployment
- 💼 Looking for my first professional experience as a full-stack developer
- 📍 Based in Tunisia · Open to remote
